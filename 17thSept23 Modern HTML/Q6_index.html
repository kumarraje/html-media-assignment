Comparison between HTML and HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 is the fifth revision of HTML.
HTML is mainly used for structuring content on the web. HTML5 introduces new elements and attributes for better structuring of web content, such as <header>, <nav>, <footer>, <video>, and <audio>.
HTML4 has limited support for multimedia content. HTML5 provides native support for multimedia elements like audio and video, reducing the need for third-party plugins like Flash.
